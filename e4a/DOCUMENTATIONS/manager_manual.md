# EduTech Manager Manual

**Role:** School Manager  
**Access Level:** School-wide management  
**Primary Responsibilities:** Manage classes, students, teachers, parents, and academic terms

---

## Getting Started

### Initial Setup

**After Admin Approval:**
1. You'll receive confirmation that your school is approved
2. Log in at: http://your-domain.com/login
3. You'll see the **Manager Dashboard**

### Dashboard Quick Actions

Your dashboard has 7 main cards:
1. **Manage Classes** (Green)
2. **Manage Students** (Cyan)
3. **Link Parents** (Purple)
4. **Academic Terms** (Indigo)
5. **Manage Teachers** (Blue)
6. **Announcements** (Yellow)
7. **School Status** (Shows approval status)

## Managing Classes

### Create a New Class

**Step 1:** Click **"Manage Classes"**

**Step 2:** Click **"Create New Class"**

**Step 3:** Fill in the form:
- **Class Name:** e.g., "6ème A", "CM2 B"
- **Level:** Select from dropdown
- **Academic Year:** Auto-filled (current year)
- **Capacity:** Maximum number of students

**Step 4:** Click **"Create Class"**

✅ **Success!** Class is now available for student enrollment

### Edit a Class

1. Go to **Manage Classes**
2. Find the class in the table
3. Click **"Edit"**
4. Update information
5. Click **"Update Class"**

### Delete a Class

⚠️ **Warning:** Can only delete empty classes

1. Ensure no students are enrolled
2. Click **"Delete"** next to the class
3. Confirm deletion

---

## Managing Students

### Add a New Student

**Step 1:** Click **"Manage Students"**

**Step 2:** Click **"Add New Student"**

**Step 3:** Fill in student information:
- **Full Name:** Student's complete name
- **Email:** For student login (must be unique)
- **Date of Birth:** Use date picker
- **Class:** Select from available classes
- **Address:** Optional but recommended

**Step 4:** Click **"Create Student"**

**IMPORTANT:** 
- A password will be **auto-generated**
- **Save the password** - it's shown only once!
- Give this password to the student/parent

### Assign Student to Class

During creation:
- Select class from dropdown

After creation:
- Go to student's edit page
- Change the class
- Click "Update"

### View Student List

**Filter Options:**
- All students
- By class
- Search by name

**Information Shown:**
- Student name
- Class
- Email
- Enrollment date

## Parent Management

### Create Parent Account

**Step 1:** Click **"Link Parents"**

**Step 2:** Click **"Create New Parent"**

**Step 3:** Fill in parent information:
- **Name:** Parent's full name
- **Email:** Unique email for login
- **Phone:** Contact number (optional)
- **Address:** Home address (optional)

**Step 4:** Click **"Create Parent"**

**CRITICAL:** 
- ✅ Auto-generated password is displayed **once only**
- ✅ **Copy and save it** immediately
- ✅ Share with the parent securely

### Link Student to Parent

**Step 1:** From Parents list, click **"Link Student"**

**Step 2:** Parent information is shown

**Step 3:** Under "Link New Student":
- Select student from dropdown
- Only **unlinked students** show

**Step 4:** Click **"Link Student"**

### Multiple Children

One parent can have **multiple children**:
1. Link first child (as above)
2. Click "Link Student" again
3. Select second child
4. Repeat for additional children

### Unlink a Student

1. Go to parent's link page
2. Find student in "Current Children"
3. Click **"Unlink"**
4. Confirm action

## Academic Terms Management

### Why Use Terms?

Terms (semesters/trimesters) let you:
- Organize the academic year
- Generate per-term report cards
- Track progress over time
- Assign grades to specific periods

### Create Academic Terms

**Step 1:** Click **"Academic Terms"** card

**Step 2:** Click **"Create New Term"**

**Step 3:** Fill in information:
- **Academic Year:** e.g., "2024-2025"
- **Term Name:** e.g., "1st Semester", "Trimester 1"
- **Start Date:** First day of term
- **End Date:** Last day of term
- **Order:** Display order (1, 2, 3...)
- **Mark as Current:** Check if this is the active term

**Step 4:** Click **"Create Term"**

### Common Term Structures

**2 Semesters:**
- 1st Semester: September - January
- 2nd Semester: February - June

**3 Trimesters:**
- 1st Trimester: September - December
- 2nd Trimester: January - March
- 3rd Trimester: April - June

### Set Current Term

**Important:** Only one term can be current at a time

1. Go to Academic Terms
2. Find the term
3. Click **"Set Current"**
4. Previous current term automatically becomes inactive

**Effect:** New grades are automatically assigned to the current term

## Common Issues

### "Can't create parent - email already exists"
✅ That email is already used. Try a different email or check if parent account exists

### "Student not showing in dropdown to link"
✅ Student may already be linked to another parent. Check student's details

### "Teacher can't enter grades"
✅ Verify teacher is assigned to the correct class and subject

### "Report cards don't show term data"
✅ Ensure academic terms are created and current term is set

## Tips for Success

**Organization:**
- Create all classes at the start of year
- Use consistent naming (e.g., "6ème A", "6ème B")
- Keep parent contact info updated

**Communication:**
- Securely share passwords (email, sealed envelope)
- Post announcements for important updates
- Respond to parent inquiries promptly

**Data Management:**
- Regularly back up important information
- Review student enrollments each term
- Update class capacities as needed
